Javascript é un linguaggio di programmazione che ha conosciuto negli ultimi tempi uno sviluppo e una diffusione senza precedenti. Nato come linguaggio di programmazione web, cioé da eseguire all'interno di un browser, con il tempo si é diffuso e ora viene utilizzato non solo per produrre applicazioni da utilizzare in questo ambito, ma anche per realizzare applicazioni per telefoni cellulari e desktop, cioé da utilizzare nel vostro sistema operativo preferito, Windows, Mac o Linux. Per seguire questa guida introduttiva non servono conoscenze particolari, solo basi di HTML e CSS e la voglia di imparare a programmare. E' possibile scrivere codice Javascript con qualunque editor di codice. Il consiglio é provare Visual Studio Code, disponibile a questo link: https://code.visualstudio.com/ Fatto questo, per scrivere codice Javascript basta inserire in un normale file HTML il tag script: tutto quello che si trova all'interno sarà interpretato come codice Javascript. 
  
    <script>

    .....

    .....

    .....

    </script>
Abbiamo capito come inserire codice js nella nostra pagina, vediamo ora le basi del linguaggio. Anzitutto, impariamo ad utilizzare le variabili in js. Per dichiarare una variabile in js, utilizziamo il comando var seguito dal nome della variabile.
IMPORTANTE:
 Prima di utilizzare una variabile per la prima volta, questa deve essere dichiarata! Se non lo farete, Javascript vi restituirà un errore, variabile non definita!
Ecco un esempio di come dichiarare una variabile in Javascript:
    
        var pippo;
E' possibile anche assegnare subito un valore alla variabile:
    
        var pippo = 10;
Come in tutti i linguaggi di programmazione, anche in Javascript le variabili sono caratterizzate da un tipo. Le variabili in JS possono essere numeri, stringhe, boolean (possono avere solo due valori, vero e falso), null (che significa nessun valore) e undefined (valore non definito).
E' possibile assegnare subito un numero ad una variabile:
    
        var pippo = 10;
Ed é possibile assegnare una stringa contenente lo stesso numero alla stessa variabile:
    
        var pippo = "10";
In questo caso la differenza consiste nelle virgolette, che "spiegano" alla macchina come deve considerare il contenuto della variabile, se numero o lettera,
IMPORTANTE:
 Spesso proprio i tipi di variabili sono causa di errori. Ad esempio, l'operatore + serve sia per unire tra loro diverse stringhe sia per unire diverse stringhe tra loro sia per fare le somme! Ad esempio: 
    
        var elemento1 = 10;
        var elemento2 = 20;
        var somma = elemento1 + elemento2;
In questo caso somma avrà valore 30.
Al contrario in questo caso: 
    
        var elemento1 = "10";
        var elemento2 = "20";
        var somma = elemento1 + elemento2;
Somma avrà come valore la stringa "1020". Verranno infatti unite due stringhe (in gergo tecnico, verrà effettuata la concatenazione di due stringhe). In Javascript si possono anche creare array (o vettori, oppure liste per chi viene da Python). Un esempio: 
    
        var giorniDellaSettimana = [
	        "lunedì",
	        "martedì",
	        "mercoledì",
	        "giovedì",
	        "venerdì",
	        "sabato",
	        "domenica"];
Come vedete, le liste funzionano come le altre variabili, con una particolarità: l'elenco si apre e si chiude con le parentesi quadre [], e ogni variabile inserita é separata dalle altre dalla virgola.
Per accedere ad una singola posizione, basta scrivere nomevariabile[posizione]. Nel nostro esempio, per accedere al giorno martedì basta scrivere giorniDellaSettimana[2].
IMPORTANTE:
 La posizione di un elemento all'interno di un array é detta indice. RICORDA: LA PRIMA POSIZIONE E' ZERO, NON 1!!!!!

IF e condizioni

Come tutti i linguaggi di programmazione che si rispettino, anche js utilizza le "istruzioni condizionali": un blocco di codice viene eseguito solo se si verifica una determinata condizione. Rendiamo tutto più chiaro con un esempio: 
    
        var pippo = 10;
        if (pippo > 5) {
	        var pluto = "test";
        }
In questo caso la condizione é verificata, e il codice tra le parentesi verrà eseguito, e verrà assegnato alla variabile pluto il valore "test".
E' possibile anche inserire un blocco di codice da eseguire qualora la condizione NON sia verificata, tramite l'istruzione ELSE. Vediamo un esempio: 
    
        var pippo = 10;
        if (pippo < 5) {
	        var pluto = "valore1";
        } else {
            var pluto = "valore2";
        }
In questo caso la condizione non é verificata, e la variabile assumerà il valore "valore2".
Ora prova tu!!!!
    
        var pippo = "indovinello";
        if (pippo == "indovinello") {
	        var pluto = 10;
        } else {
            var pluto = 20;
        }
Che valore ha pluto?
In Javascript é anche possibile combinare diverse condizioni con gli operatori && (che corrispondono a and, per chi ha già incontrato questo operatore in altri linguaggi) e || (si possono ottenere su Windows con "shift" + "\", e corrispondono ad or).
Ad esempio: 
    
        var pippo = 10;
        if (pippo < 5 && pippo < 12) {
	        var pluto = "valore1";
        } else {
            var pluto = "valore2";
        }
In questo caso solo la prima condizione é verificata, dunque la variabile ha valore "valore2", ossia viene eseguito il blocco di codice contenuto nell'else.
Ma in questo caso: 
    
        var pippo = 10;
        if (pippo < 5 || pippo < 12) {
	        var pluto = "valore1";
        } else {
            var pluto = "valore2";
        }
La variabile ha valore "valore1", perché ALMENO una delle due condizioni é verificata.
In questo caso veniva controllata una sola condizione. Ci sono pero' casi in cui diventa necessario controllarne diverse. In questo caso si può utilizzare l'istruzione switch...case. Vediamola insieme:
    
    var pippo = 7;
    switch (pippo) {
	    case 1:
		    pluto = 1;
	        break;
	    case 7:
		    pluto = 2;
	        break;
	    // ...
	    default:
            pluto = 3;
	        break;
    }
In questo caso pluto avrà valore 2. La variabile viene infatti confrontata con le condizioni contenute nei diversi "case", e quando incontra quello "giusto", viene eseguito il codice contenuto.
Può però accadere che nessuna delle condizioni sia verificata. In questo caso, viene eseguito il blocco di istruzioni contenuto in "default". Ad esempio: 
    
    var pippo = 10;
    switch (pippo) {
	    case 1:
		    pluto = 1;
	        break;
	    case 7:
		    pluto = 2;
	        break;
	    // ...
	    default:
            pluto = 3;
	        break;
    }
In questo caso, pluto avrà valore 3.
Ora prova tu!!!!
    
    var pippo = 15;
    switch (pippo) {
	    case 3:
		    pluto = 1;
	        break;
	    case 10:
		    pluto = 2;
	        break;
	    // ...
	    default:
            pluto = 3;
	        break;
    }
Che valore ha pluto?

I cicli

Non bastano però gli "IF" a definire un insieme di istruzioni un linguaggio di programmazione, servono anche i cicli!
Vediamone alcuni.
Il primo é il ciclo "while": le istruzioni vengono eseguite fino a quando la condizione é verificata. Un esempio: 
    
        var y = 1;
        var i = 1;
        while (i <= 4) {
	        y = y + 2;
	        i++;
        }
In questo caso, l'operazione viene eseguita 4 volte (con l'istruzione i++ il valore di i viene aumentato ogni volta di 1), e alla fine y avrà valore 9.
In questo secondo caso, però, l'operazione non viene mai eseguita, perché la condizione non é mai verificata, e y rimane uguale a 1: 
    
        var y = 1;
        var i = 5;
        while (i <= 4) {
	        y = y + 2;
	        i++;
        }
C'é però un altro tipo di ciclo, che prevede l'operazione sia SEMPRE eseguita almeno una volta, anche qualora la condizione non sia mai verificata. é il DO WHILE. Vediamo la sintassi: 
    
        var y = 1;
        var i = 5;
        do {
	        y = y + 2;
	        i++;
        }
        while (i <= 4)
In questo secondo caso, alla fine y avrà valore 3. Ora prova tu!!!!
    
        var y = 10;
        var i = 1;
        while (i < 4) {
	        y = y + 10;
	        i++;
        }
Che valore ha pluto?
L'ultimo tipo di ciclo che vediamo é il ciclo for, che ha la seguente sintassi:
    
        for (inizializzazione; condizione; modifica) {
	        // istruzioni
        }
Un piccolo esempio renderà tutto più chiaro: 
    
        var y = 0;
        for (i = 0; i < 4; i++) {
	        Y = y + 1;
        }
In questo caso, alla fine y avrà valore 4. L'operazione verrà infatti eseguita 4 volte. La variabile i avrà valore iniziale 0, e verrà incrementata ogni volta fino a quando il suo valore sarà inferiore a 4. A quel punto il ciclo avrà termine. Ora prova tu!!!!
    
        var y = 0;
        for (i = 0; i < 10; i++) {
	        Y = y + 10;
        }
Che valore finale avrà y?
Il ciclo for in Javascript offre una altra opportunità fondamentale. E' infatti possibile utilizzarlo per scorrere tutti gli elementi di un array (o lista). Un esempio renderà tutto più chiaro:
    
        var quantita = [12, 34, 45, 7, 19]; // definisco una variabile quantità, che é un array (o lista).
        var totale = 0; // definisco una seconda variabile, destinata a contenere la somma di tutti gli elementi della lista.
        var indice; 
        // Definisco una variabile denominata indice. A cosa serve? Semplice. Con il ciclo for in, la cui sintassi é for (variabileindice in variabilelista) {} 

        // Javascript farà scorrere tutti gli elementi di una lista, assegnando di volta in volta alla variabile indice la posizione che un dato elemento occupa.
        // E' quindi molto semplice ottenere tutti gli elementi uno alla volta, basta estrarre passo per passo l'elemento della lista in posizione indice.
        for (indice in quantita) {
	        totale = totale +  quantita[indice];
        }
Se non mi interessa la posizione di un elemento nella lista ma solo il suo valore, posso usare il ciclo for ... of. 
    
        var quantita = [12, 34, 45, 7, 19];
        var totale = 0;
        var valore;
        for (valore of quantita) {
	        totale = totale +  valore;
        }
In entrambi i casi, alla fine del ciclo la variabile totale assumerà identico valore. Come avrete notato, ho inserito dei commenti senza ricevere alcune messaggio di errore. Javascript interpreta infatti tutto quello che segue la doppia barra // come un commento.
Se si vogliono fare commenti di più righe, la sintassi é la seguente: /* apre il commento. Per chiuderlo basta scrivere */ A volte si rende necessario interrompere un ciclo al verificarsi di determinate condizioni. E' possibile farlo con l'istruzione break. Ad esempio:
    
        var x=0;
        while (1 == 1) {
	        console.log(x);
	        // condizione di uscita
	        if (x > 100) break;
            x++;
        }
In questo caso, il ciclo continuerebbe in eterno (la condizione 1 = 1 é sempre verificata). Ma se x supera 100, il break interrompe il ciclo e provoca la fine della esecuzione.

Le funzioni

Molte volte, in Javascript come in altri linguaggi, é utile e comodo riutilizzare parti di codice, ad esempio per compiti che devono essere svolti spesso con variabili diverse. E' possibile farlo con le funzioni:
Ecco un esempio:
    
        function somma(x, y) {
            var z = x + y;
            return z;
        }
        var totale = somma(2,3);
In questo caso, viene definita una funzione somma e tra la parentesi rotonda vengono definite due variabili, x e y. Quando la funzione viene richiamata attraverso l'istruzione somma(2,3)
la prima variabile tra parentesi, ossia la variabile x, acquisisce il primo valore passato, ossia 2, mentre la seconda variabile, la variabile y, acquisisce il valore 3.
All'interno della funzione vengono poi eseguite operazioni, e la funzione restituisce il valore della variabile z tramite l'istruzione return z. A questo punto, la variabile totale acquisirà il valore della variabile restituita dalla funzione somma, ossia z.
E' possibile anche assegnare alla funzione dei valori di default, cioé dei valori che le variabili avranno qualora nessun valore venga passato quando la variabile viene richiamata. Vediamo un esempio: 
    
        function somma(x = 2, y = 3) {
            var z = x + y;
            return z;
        }
        var totale = somma();
In questo caso, visto che non vengono passati valori, il valore finale di totale sarà 5, cioé la somma dei due valori di default. Ora prova tu!!!!
    
        function moltiplicazione(x,y) {
            return x * y;
        }
        var totale = somma(3,5);
Che valore ha totale?
IMPORTANTE:
 Le variabili definite all'interno di una funzione non valgono al fdi fuori della stessa, Javascript vi restituirà un errore, variabile non definita!
Ad esempio: 
    
        function moltiplicazione(x,y) {
            return x * y;
        }
        var totale = somma(3,5);
        alert(x);
Restituirà valore undefined, perché la variabile x non esiste al di fuori della funzione.
Javascript fornisce una serie di funzioni predefinite, utili per ottenere i nostri scopi in tempi brevi. Ecco le principali:

isNaN e isFinite

La funzione isNaN() prende un argomento e restituisce true se il suo valore é NaN, cioé non é un valore numerico valido, false altrimenti.
isFinite() restituisce true se il valore del suo argomento é diverso da Infinity (infinito) e da NaN.

parseInt e parseFloat

Convertono una stringa rispettivamente in un numero intero e in un numero decimale.

Gli oggetti

Le variabili più complesse che abbiamo visto finora sono gli array: elenchi di valori caratterizzati ognuno da una posizione e da un valore. C'é però un tipo di variabile più complesso ma molto utile ed utilizzato nella programmazione moderna: gli oggetti.
Ogni oggetto conterrà dati, detti proprietà e rappresentati da coppie di nome-valore, e funzionalità, dette metodi e rappresentate da funzioni.
Un esempio renderà tutto più chiaro. Supponiamo di voler memorizzare i nostri dati, ossia nome, cognome, e data di nascita. Potremmo usare una lista:
    
        var dati = ["Pierluigi","Zavaroni","28/3/1977"];
Sebbene possibile, una simile conservazione é scomoda, perché risulterebbe be presto impossibile (o comunque difficile) risalire a quale dato é il nome, quale il cognome e quale la data di nascita. Con un oggetto é molto più semplice, vediamo come: 
    
    var dati = {
	    nome: "Pierluigi",
	    cognome: "Zavaroni",
	    nascita: "28/03/1977"
    };
Come si vede chiaramente, una sintassi molto chiara e pulita. E qualora si desideri accedere ad una delle proprietà (cioé dei dati) presenti? Anche in questo caso, la sintassi é semplice:
    
    var dati = {
	    nome: "Pierluigi",
	    cognome: "Zavaroni",
	    nascita: "28/03/1977"
    };
    alert(dati.nome);
In questo caso, riceveremo un messaggio con il nome contenuto all'interno della proprietà.
IMPORTANTE:
 Contrariamente alle liste, quando si deve associare un valore ad una proprietà di un oggetto, si utilizzano i : e non il simbolo =

Abbiamo visto però come un oggetto possa contenere anche delle funzioni, dette metodo dell'oggetto. Vediamo un esempio: 
    
    var dati = {
	    x: 2,
        y: 5,
        function somma() { return 2 + 5; }
	};
    alert(dati.somma());
In questo caso il valore stampato a video sarà 7.
E' anche possibile aggiungere proprietà o metodi ad un oggetto, tramite un procedimento detto definizione incrementale. Ecco un esempio:
    
    var dati = {
	    nome: "Pierluigi",
	    cognome: "Zavaroni",
	    nascita: "28/03/1977"
    };
    dati.eta = 43;
In questo caso, ho aggiunto una proprietà contenente il valore della mia età, 43 anni.
In molti casi nel metodo di una variabile é comodo fare riferimento al valore corrente del valore di una proprietà. In questo caso, si utilizza la parola chiave this. Ecco un esempio: 
    
    var dati = {
	    x: 2,
        y: 5,
        function somma() { return this.x + this.y; }
	};
    alert(dati.somma());
E se volessi creare un altro oggetto? Nel nostro esempio, se volessi fare la somma di nuemri diversi da 2 e 5, senza cancellare l'oggetto esistente e senza riscrivere tutto? Un oggetto può essere "duplicato" assegnandolo ad una variabile. Ecco un esempio:
    
        var datipierluigi = {
	        nome: "Pierluigi",
	        cognome: "Zavaroni",
	        nascita: "28/03/1977"
        };
        var datipietro = datipierluigi;
        var datipietro = {
	        nome: "Pietro",
	        cognome: "Rossi",
	        nascita: "21/07/1981"
        };

Il DOM

Javascript può essere utilizzato per rendere dinamiche le nostre pagine in HTML, ossia per fare in modo le stesse vengano modificate, ad esempio, dall'azione dell'utente.
Per selezionare uno o più elementi di una pagina web, allo scopo di modificarli o estrarre dati, si possono usare le istruzioni (ma sarebbe meglio dire metodi predefiniti dell'oggetto document) document.getElementById e document.getElementsByClassName. Il primo seleziona l'elemento caratterizzato da un determinato id, il secondo tutti gli elementi con una determinata classe. Facciamo un esempio:
    
        <p id="elemento1" class="classeprova"></p>
        <p id="elemento2" class="classeprova"></p>
        var p = document.getElementById('elemento1');
        var c = document.getElementsByClassName('classeprova');
In questo caso, la variabile p conterrà elemento1, mentre la variabile c conterrà una lista di due elementi, nella posizione 0 conterrà infatti il paragrafo con id elemento1 e in posizione 1 il paragrafo con id elemento2. Molto spesso un elemento di HTML (un nodo) ne contiene a sua volta altri. Succede spesso ad esempio con i div, che sono solitamente uno dentro l'altro. Javascript ci permette di vedere e accedere a tutti gli elementi (i nodi) contenuti all'interno di quello che abbiamo selezionato. Ancora una volta tutto sarà più chiaro con un esempio:
    
        <div id="DivPrincipale">
            <p>Paragrafo1</p>
            <p>Pargarafo2</p>
	        <p>Paragrafo3</p>
        </div>
        
        <script>
            var divprincipale = document.getElementById("DivPrincipale");
            for (var i = 0; i < divprincipale.childNodes.length; i++) {
	            alert(divprincipale.childNodes[i].innerHTML);
            }
        </script>
Esaminiamo il codice sopra. Prima di tutto, abbiamo un codice html con un div principale che racchiude al suo interno 3 paragrafi. Quindi con getElementById abbiamo selezionato il div con id DivPrincipale. Abbiamo poi scelto di accedere ai nodi contenuti all'interno del div principale con childNodes, che ci permette di avere un array contenente tutti i nodi racchiusi nel nodo principale. Abbiamo poi utilizzato la proprietà lenght, che contiene la lunghezza dell'array, per scorrere tutti gli elementi, andando di volta in volta a selezionare un nodo contenuto all'interno del div, tramite la sua posizione. La proprietà innerHTML ci restituisce il codice HTML contenuto all'interno del nodo. Qualora avessimo voluto cambiare il contenuto dei nodi, avremmo potuto agire nel modo seguente: 
    
        <div id="DivPrincipale">
            <p>Paragrafo1</p>
            <p>Pargarafo2</p>
	        <p>Paragrafo3</p>
        </div>
        
        <script>
            var divprincipale = document.getElementById("DivPrincipale");
            for (var i = 0; i < divprincipale.childNodes.length; i++) {
	            divprincipale.childNodes[i].innerHTML = "test";
            }
        </script>
Alla fine del ciclo, tutti e tre i paragrafi conterrebbero la parola "test".
